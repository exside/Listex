Listex Demo

Below is a demo of the Listex plugin. The first <select> is a plain Listex box with default options.

Note the disabled item in the first list. Now click on any option in the second list. The disabled item from the first list is now enabled, and all this has been bound to the standard .change() event of the second box. Easy!

The code used for this demo: 

	$(document).ready(function() {
		$('select.listex').listex();

		//$('select').find('option[value="baz"]').removeAttr('disabled');

		$('select.sel2').change(function() {
			$('select.sel1').find('option[value="baz"]').removeAttr('disabled');
		});
	});
